# 🎮 RetroArch — what is it?

**RetroArch** is an application on a jailbroken PS3 that allows you to **play dozens of retro consoles using a single program**.

👉 In simple terms:  
a **“universal console” for retro gaming**

---

# 🔥 What is it used for?

## 🕹️ 1. Emulate many consoles

With RetroArch, you can play:

- NES  
- SNES  
- Game Boy / Game Boy Advance  
- Sega Genesis  
- Game Gear  
- Atari  
- PlayStation 1  
- Arcade (MAME, FBNeo)  

👉 And much more  

---

## 🧠 2. Works with “cores”

👉 A **core = an emulator**

### Examples:

- SNES → **Snes9x**  
- NES → **FCEUmm**  
- GBA → **VBA Next**  
- Arcade → **FBNeo**  

👉 You must **load a core before launching a game**

# 📂 Important structure (PS3)

👉 Very important for it to work:  
`/dev_hdd0/game/RETROARCH/USRDIR/`

You should have:

- `cores` → emulators  
- `system` → BIOS  
- your ROMs → wherever you want (e.g. `/roms/nes`)  

---

# ⚠️ Common issues

## ❌ “0/3 scanning”

👉 Possible causes:

- Wrong folder  
- Wrong core  
- Incorrect file name  

---

## ❌ Black screen

👉 Possible causes:

- Wrong core  
- Missing BIOS  
- Incompatible game  

---

## ❌ No cheats

👉 Solution:

- Download the cheat database  
- Or add files manually
