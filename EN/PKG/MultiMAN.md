# 🎮 MultiMAN — what is it?

**MultiMAN** is the **most well-known game manager** on jailbroken PS3.  
It is usually the **first tool installed** on a PS3 running **CFW or HEN**.

---

# 🔥 What is it used for?

## 📀 1. Launch games (main function)

MultiMAN allows you to launch:

- **PS3 games** (ISO or folder format)  
- **PS1 games**  
- **PS2 games** (with some limitations)  

👉 From:

- Internal hard drive  
- USB drive  

---

## 📂 2. File manager (VERY IMPORTANT)

MultiMAN acts like a file explorer:

You can:

- Copy / paste games  
- Move files  
- Delete / rename  

Access to folders:

- `/dev_hdd0`  
- USB  
- Network  

👉 It’s like a **Windows file explorer on PS3**

---

## 🖼️ 3. Interface

- **XMB-style** (PS3 menu)  
- **Coverflow mode** (game covers)  
- Simple but slightly outdated interface  

---

## 🌐 4. Game transfer (FTP)

MultiMAN allows:

- Connection via **FileZilla**  
- Transfer of games **PC → PS3**  

👉 Very useful for installing games quickly  

---

## 📦 5. PKG installation

You can:

- Install **homebrew apps**  
- Install **PKG games**  

---

## 💿 6. Disc ripping

You can:

- Insert a game disc  
- Copy it directly to the hard drive  

👉 No need for the disc afterward
