# 🌐 webMAN MOD — what is it?

**webMAN MOD** is a **very powerful plugin** for jailbroken PS3 that allows you to  
**launch your games directly from the PS3 menu (XMB)** without opening any application.

👉 In simple terms:  
everything is **integrated directly into the console**

---

# 🔥 What is it used for?

## 🎮 1. Launch your games instantly

Compatible with:

- **PS3 games** (ISO / folder)  
- **PS2 / PS1 / PSP games**  

👉 They appear in: Game → webMAN Games  

✔️ Click → mount → launch  

---

## ⚡ 2. Ultra fast

👉 Unlike MultiMAN:

- No long loading times  
- No complicated navigation  

---

## 🌡️ 3. Fan control (VERY IMPORTANT)

webMAN allows you to:

- Control fan speed  
- Prevent overheating 🔥  

You can:

- Use automatic mode  
- Set manual speed  

👉 Recommendation:

- Keep CPU around **60–68°C**

---

## 🌐 4. Web interface (PC / phone)

You can access your PS3 via browser:  
http://192.168.X.X

You can:

- Launch games  
- Check temperatures  
- Turn off the console  
- Manage files  

---

## 📂 5. Used folders

webMAN automatically detects:

- /dev_hdd0/PS3ISO/  
- /dev_hdd0/PS2ISO/  
- /dev_hdd0/PSXISO/  
- /dev_hdd0/PSPISO/  

👉 Put your games there → they appear automatically  

---

## 🧠 6. Automatic scan

- Automatically detects games  
- Updates the list  
- No setup required
