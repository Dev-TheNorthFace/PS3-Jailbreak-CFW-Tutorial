# 🎮 Apollo Save Tool (PS3)

**Apollo Save Tool** is a very powerful tool for jailbroken PS3 (**CFW or HEN**).  
It allows you to **manage, modify, and import game saves** easily.

---

# 🔧 Main features

## 💾 1. Modify saves (cheats)

You can:

- Add money, levels, or items  
- Unlock content  
- Modify stats  

👉 Examples:

- **GTA V** → add money  
- **Dark Souls** → modify your character  
- **Call of Duty** → campaign stats  

---

## 🔓 2. Unlock protected saves

Some games lock their saves (linked to a PSN account).

Apollo allows you to:

- Remove protection  
- Use downloaded saves  

---

## 🔁 3. Import / export saves

You can:

- Copy saves from USB → PS3  
- Backup PS3 saves → USB  
- Install downloaded saves  

---

## 🧬 4. Resign a save

Very important 👇  

A save from another user **does not work directly**.

Apollo allows you to:

- Adapt the save to your PSN account  

👉 This is called **resigning**

---

## 🛠️ 5. Repair saves

- Fix some corrupted saves  
- Recover broken progress  

---

## 🌐 6. Download saves online

Directly in Apollo:

- Download modified saves  
- 100% completed saves  
- Saves with cheats  

---

# ⚠️ Important

- ❌ Avoid online → risk of PSN ban  
- ✔️ Best for:
  - Single-player  
  - Testing  
  - Fun  
  - Quickly unlocking content  

---

# 💡 Quick example

You want a 100% save:

1. Put the save on your USB drive  
2. Launch **Apollo Save Tool**  
3. Do:

   - **Import Save**  
   - **Resign**  

4. Copy the save to the PS3  

👉 Result: game completed at 100%
