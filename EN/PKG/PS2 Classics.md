# 🎮 PS2 Classics — what is it?

**PS2 Classics** is a method on a jailbroken PS3 that allows you to **play PS2 games converted into a special format compatible with Sony’s official emulator**.

👉 In simple terms:  
you convert a **PS2 game → into an “official PS3 game”**

---

# 🔥 What is it used for?

## 🕹️ 1. Play PS2 games on PS3

- Compatible with almost all PS3 models (even non-backward compatible ones)  
- Works with **CFW** and **HEN**  

👉 Unlike older PS3 FAT models that could read real PS2 discs  

---

## 📦 2. PKG format (very convenient)

PS2 Classics games are:

- Converted into **.PKG files**  
- Installed like a normal game  

👉 They appear directly in the PS3 menu  

---

# ⚙️ How does it work?

## 🔄 1. Conversion

You take:

- A PS2 game in **ISO format**

You use a tool like:

- **PS2 Classics GUI**

👉 It creates:

- An **installable PKG file**

---

## ▶️ 2. Launching

- Install the PKG  
- Launch the game like a PS3 game  

---

## 🧠 3. Required encryption

⚠️ Very important:

Games must be:

- **Encrypted**  

Otherwise:

- ❌ Black screen  
- ❌ Crash
