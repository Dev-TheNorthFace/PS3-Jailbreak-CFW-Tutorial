# 🔥 ManaGunZ — what is it?

**ManaGunZ** is an **advanced game manager** for jailbroken PS3 (**mainly CFW**).  
It is especially known for its **automatic PS2 patching** and advanced tools.

---

# 🧠 What is it used for?

## 🎮 1. Game management (PS1 / PS2 / PS3)

ManaGunZ allows you to:

- Launch **PS3 games (ISO / folder)**  
- Launch **PS2 games (ISO)**  
- Launch **PS1 games**  

👉 An alternative to **multiMAN**, but more technical

---

## 🧠 2. Automatic PS2 patches (BIG ADVANTAGE)

👉 This is where ManaGunZ really shines 🔥

It allows you to:

- Automatically apply **PS2 compatibility configs**
- Fix:
  - Graphical bugs  
  - Black screens  
  - Performance issues  

---

## ⚙️ 3. Modify PS2 configurations

You can also:

- Force resolution  
- Enable / disable certain patches  
- Test different configurations  

👉 Useful for PS2 games that are difficult to run

---

## 📦 4. Advanced file management

- Built-in file browser  
- Copy / paste games  
- Manage ISO folders  

---

## 🎨 5. Customizable interface

- Automatic cover art  
- Clean and modern interface  
- Better game organization  

---

## 🧪 6. Advanced tools (for advanced users)

- Game analysis  
- Module management  
- Debug options  

---

# ⚠️ Important

- ✔️ Works best on **CFW**  
- ⚠️ Less simple than multiMAN (more technical)  
- ✔️ Ideal for:
  - PS2 games  
  - Advanced compatibility  
  - Experienced users  

---

# 💡 Example

A PS2 game doesn’t work:

1. Launch the game with ManaGunZ  
2. It automatically applies a config  
3. If it still doesn’t work:
   - Adjust the patches manually  

👉 And often → the game works
