# 🛠️ Step 2: Prepare the browser and use PS3 Toolset

---

## 🌐 1. Open the internet browser

- Go to **Network → Internet Browser**

---

## 🔧 2. Reset the browser (IMPORTANT)

Press **🔺 (Triangle)** → **Tools**, then configure:

- **Cookies** → Allow  
- **JavaScript** → Enabled  
- **Confirm Browser Close** → Off  
- **Home Page** → Use Default Page  

Then:

- **Delete Cookies** → Yes  
- **Clear Search History** → Yes  
- **Clear Cache** → Yes  
- **Clear Authentication Information** → Yes  

👉 Fully close the browser, then **reopen it**

---

## 🔎 3. Access the PS3 Toolset website

1. Press **Start**  
2. Enter the address: **ps3toolset.com**  
3. Start the search  

---

## ⚠️ If the message “Update your browser” appears

Check:

1. The address is correctly typed (no extra spaces)  
2. You used the browser address bar, not **Start**

👉 Otherwise, redo the browser reset

---

## ⏳ 4. Website loading

- The site may take **1 to 2 minutes** to load  
- Messages may appear at the top right → **this is normal**  

👉 Do not touch anything while it loads  

---

## 📜 5. Allow scripts

- If the browser asks to install a script → **Yes**  
- If a crypto donation message appears → **No**  

---

## 🔁 If the site does not work

- Close the browser  
- Perform a **full browser reset again**  
- Try again  

---

## 🧠 6. Check CFW compatibility

1. Click on **System Manager**  
2. Check:

- ✅ **CFW Compatible PS3** → Compatible  
- ❌ **Red cross** → Not compatible  

---

## 💾 7. Backup flash memory (VERY IMPORTANT)

1. Plug a USB drive into the **rightmost port** (near the disc drive)  
2. Click:

- **Flash Memory → Save Flash Memory Backup**  
- Select **dev_usb000**  
- Click **Save**

👉 Wait until it finishes, then click **Close**

📌 A file named **dump.bin** will be created on your USB  
👉 Keep it safe (backup in case of problems)

---

## ⚙️ 8. Apply the CFW patch

### 1. Load the patch

- **Flash Memory Patch → Load Patch via HTTP**  
- Wait → Click **Close**

### 2. Apply the patch

- **Flash Memory Patch → Apply Loaded Patch**

⚠️ Confirmation message:

> You are responsible for your console.  
> A wrong action may cause issues.

👉 Click **Yes**

- Wait for the process to finish  
- Click **Close**

---

## ✅ 9. End of step

- Exit the browser  

---

# 🎉 Step 2 complete!

👉 You can now move on to **Step 3**
