# 🎮 1. Understanding the PKG Manager

In the **Game** tab, you will find:

- 📁 **PlayStation 3** (not needed here)  
- 📁 **Package Manager** (IMPORTANT)  

---

## 📦 What is a PKG file?

A **.PKG** file is a format used by the PS3 to:

- Install games  
- Install applications (homebrew)  
- Install tools  

👉 It’s like an **.exe file on Windows**

---

## 📂 Inside "Package Manager"

You will find:

### 📁 PlayStation Network Content

- Used for certain specific content  
- 👉 Not important for us here  

---

### 📁 Install Package Files

👉 The most important section

In this folder:

- **Standard** → install PKG files from USB ✅  
- Other options → less useful  

---

### 🗑️ Delete Package Files

- Allows you to delete installed PKG files

# 🎨 3. Install a PS3 theme

## 📥 1. Download a theme

- Required format: `.p3t`  

👉 Example website:  
http://www.ps3-themes.com  

---

## 📂 2. Prepare the USB drive

On your USB drive, create:

PS3/  
└── THEME/  
  └── NOMDUTHEME.p3t  

---

## 🔌 3. Install on the PS3

1. Plug in the USB (rightmost port)  
2. Go to: Theme Settings → Theme  
3. Select:

- **Install**  
- **USB (usb000)**  

4. Choose your theme  
5. Click **Apply**
