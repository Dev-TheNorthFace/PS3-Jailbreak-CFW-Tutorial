# 🚀 Step 1: Console Setup

## 🔍 Check system version

When you first access your console:

👉 Go to:  
**Settings → System Settings → System Information**

👉 Check the system software version installed on your PlayStation 3.  

✔️ Recommended latest version: **4.92**

---

## 🌐 Internet setup

Next, configure your console’s internet connection.

👉 If your console cannot connect to the internet, there may be **3 reasons**:

### 1. Hardware issue ❌
The console may be defective and unable to connect to the internet.  
👉 Unfortunately, it is not possible to jailbreak a console without internet access.

---

### 2. Network security issue ⚠️
Your network may not be compatible (required format: **WPA2-PSK (AES)**).

👉 Solution:
- Use a **mobile hotspot** from your phone (LTE / 4G / 5G)

---

### 3. Incorrect date and time ⏰
If your console is set to an old date (e.g., 2012), it won’t connect to current servers.

👉 Solution:
- Go to **Settings → Date and Time**
- Set the date via the internet

📌 If it still doesn’t work:
- Try setting the date and time manually  
- Your firmware is probably outdated  
- Update to **version 4.92**

---

## ⚙️ Important settings

After setting up the internet:

👉 Go to:  
**System Settings**

- Disable **Automatic Update** ❌

👉 Then:
- Go back to **Date and Time**
- Set the time via the internet (if not already done)

---

## ✅ End of step

🎉 That’s it, step one is complete!  
👉 Proceed to **step two**.
