# 🛠️ 第二步：准备浏览器并使用 PS3 Toolset

---

## 🌐 1. 打开互联网浏览器

- 前往 **网络 → 互联网浏览器**

---

## 🔧 2. 重置浏览器（重要）

按 **🔺（三角）** → **工具**，然后设置：

- **Cookies（Cookie）** → 允许  
- **JavaScript** → 启用  
- **确认关闭浏览器** → 否  
- **主页** → 使用默认页面  

然后执行：

- **清除 Cookies** → 是  
- **删除搜索历史** → 是  
- **清除缓存** → 是  
- **清除认证信息** → 是  

👉 完全关闭浏览器，然后**重新打开**

---

## 🔎 3. 访问 PS3 Toolset 网站

1. 按 **Start**  
2. 输入地址：**ps3toolset.com**  
3. 开始搜索  

---

## ⚠️ 如果出现“请更新浏览器”的提示

请检查：

1. 地址是否正确（没有多余空格）  
2. 是否使用了工具栏搜索，而不是 **Start**

👉 如果仍然有问题，请重新执行浏览器重置

---

## ⏳ 4. 网站加载

- 网站可能需要 **1 到 2 分钟**加载  
- 右上角会显示提示信息 → **这是正常的**  

👉 加载过程中不要操作  

---

## 📜 5. 允许脚本

- 如果浏览器提示安装脚本 → **选择 是**  
- 如果出现加密货币捐赠提示 → **选择 否**  

---

## 🔁 如果网站无法使用

- 关闭浏览器  
- 重新进行**完整的浏览器重置**  
- 再次尝试  

---

## 🧠 6. 检查 CFW 兼容性

1. 点击 **System Manager**  
2. 查看：

- ✅ **CFW Compatible PS3** → 支持  
- ❌ **红色叉号** → 不支持  

---

## 💾 7. 备份闪存（非常重要）

1. 将 USB 插入 **最右侧端口**（靠近光驱）  
2. 点击：

- **Flash Memory → Save Flash Memory Backup**  
- 选择 **dev_usb000**  
- 点击 **Save**

👉 等待完成，然后点击 **Close**

📌 会在 USB 上生成一个 **dump.bin** 文件  
👉 请妥善保存（用于出现问题时恢复）

---

## ⚙️ 8. 应用 CFW 补丁

### 1. 加载补丁

- **Flash Memory Patch → Load Patch via HTTP**  
- 等待 → 点击 **Close**

### 2. 应用补丁

- **Flash Memory Patch → Apply Loaded Patch**

⚠️ 会出现确认提示：

> 您需要对您的主机负责。  
> 错误操作可能导致问题。

👉 点击 **Yes**

- 等待过程完成  
- 点击 **Close**

---

## ✅ 9. 步骤完成

- 退出浏览器  

---

# 🎉 第二步完成！

👉 现在可以进入 **第三步**
