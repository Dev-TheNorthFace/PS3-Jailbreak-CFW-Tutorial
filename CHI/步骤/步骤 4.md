# 🛠️ 第四步：安装工具 + 个性化 PS3

---

# 🎮 1. 了解 PKG 管理器

在 **Game（游戏）** 选项卡中，你会看到：

- 📁 **PlayStation 3**（这里不需要）  
- 📁 **Package Manager（PKG 管理器）**（重要）  

---

## 📦 什么是 PKG 文件？

**.PKG** 文件是 PS3 使用的一种格式，用于：

- 安装游戏  
- 安装应用程序（homebrew）  
- 安装工具  

👉 类似于 Windows 上的 **.exe 文件**

---

## 📂 在“Package Manager”中

你会看到：

### 📁 PlayStation Network 内容

- 用于某些特定内容  
- 👉 对我们来说不重要  

---

### 📁 安装 PKG 文件

👉 最重要的部分

在此文件夹中：

- **Standard** → 从 USB 安装 PKG ✅  
- 其他选项 → 不太重要  

---

### 🗑️ 删除 PKG 文件

- 用于删除已安装的 PKG 文件  

---

# 📥 2. 下载必要的软件

在 **Software（GitHub）** 文件夹中，点击 **Liens.md** 文件，然后点击链接下载 `.pkg` 文件：

- Apollo Save Tool  
- ManaGunZ  
- MultiMAN  
- PS2 Classics  
- PS2CONFIG  
- RetroArch  
- ROMS PKG  
- webMAN MOD  

---

## 🧠 为什么需要这些工具？

| 软件 | 作用 |
|------|------|
| Apollo | 修改存档 |
| ManaGunZ | 高级 PS2 支持（兼容性） |
| MultiMAN | 基础管理器 |
| PS2 Classics | PS2 游戏（PKG 格式） |
| PS2CONFIG | 修复 PS2 问题 |
| RetroArch | 复古模拟 |
| ROMS PKG | 简单的复古游戏 |
| webMAN MOD | 启动游戏 + 风扇控制 |

# 🎨 3. 安装 PS3 主题

## 📥 1. 下载主题

- 必须格式：`.p3t`  

👉 示例网站：  
http://www.ps3-themes.com  

---

## 📂 2. 准备 USB

在 USB 中创建：

PS3/  
└── THEME/  
  └── NOMDUTHEME.p3t  

---

## 🔌 3. 在 PS3 上安装

1. 插入 USB（最右侧端口）  
2. 前往：主题设置 → 主题  
3. 选择：

- **安装**  
- **USB（usb000）**  

4. 选择你的主题  
5. 点击 **应用**  

---

- ✔️ 已配置  
- ✔️ 可以正常使用
