# 🎮 Apollo Save Tool（PS3）

**Apollo Save Tool** 是一款非常强大的 PS3 越狱工具（支持 **CFW 或 HEN**）。  
它可以让你轻松**管理、修改和导入游戏存档**。

---

# 🔧 主要功能

## 💾 1. 修改存档（作弊）

你可以：

- 添加金钱、等级、物品  
- 解锁内容  
- 修改游戏数据  

👉 示例：

- **GTA V** → 增加金钱  
- **Dark Souls** → 修改角色  
- **Call of Duty** → 修改战役数据  

---

## 🔓 2. 解锁受保护的存档

有些游戏的存档是绑定 PSN 账号的。

Apollo 可以：

- 移除保护  
- 使用从网上下载的存档  

---

## 🔁 3. 导入 / 导出存档

你可以：

- 从 USB → PS3 复制存档  
- 将 PS3 存档备份到 USB  
- 安装下载的存档  

---

## 🧬 4. 重新签名存档（Resign）

非常重要 👇  

其他玩家的存档**不能直接使用**。

Apollo 可以：

- 将存档绑定到你的 PSN 账号  

👉 这一步叫做 **Resign**

---

## 🛠️ 5. 修复存档

- 修复损坏（corrupted）的存档  
- 恢复损坏的游戏进度  

---

## 🌐 6. 在线下载存档

在 Apollo 中可以直接：

- 下载修改过的存档  
- 100% 完成存档  
- 带作弊的存档  

---

# ⚠️ 重要

- ❌ 不要用于联机 → 有封号风险  
- ✔️ 适合用于：
  - 单人模式  
  - 测试  
  - 娱乐  
  - 快速解锁游戏内容  

---

# 💡 快速示例

如果你想使用 100% 存档：

1. 将存档放到 USB  
2. 打开 **Apollo Save Tool**  
3. 操作：

   - **Import Save（导入存档）**  
   - **Resign（重新签名）**

4. 将存档复制到 PS3  

👉 结果：游戏直接 100% 完成
