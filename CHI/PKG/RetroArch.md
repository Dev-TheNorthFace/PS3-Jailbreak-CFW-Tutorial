# 🎮 RetroArch — 是什么？

**RetroArch** 是一款在已破解（jailbreak）的 PS3 上运行的应用程序，可以**通过一个软件模拟几十种复古游戏主机**。

👉 简单来说：  
一个用于复古游戏的**“通用游戏机”**

---

# 🔥 有什么作用？

## 🕹️ 1. 模拟多种主机

使用 RetroArch，你可以玩：

- NES  
- SNES  
- Game Boy / Game Boy Advance  
- Sega Genesis  
- Game Gear  
- Atari  
- PlayStation 1  
- 街机（MAME, FBNeo）  

👉 以及更多

---

## 🧠 2. 使用“核心（cores）”运行

👉 一个 **core = 一个模拟器**

### 示例：

- SNES → **Snes9x**  
- NES → **FCEUmm**  
- GBA → **VBA Next**  
- 街机 → **FBNeo**  

👉 在启动游戏之前，你需要**先加载一个 core**

---

# ⚙️ 3. 强大的功能

## ⏪ 即时存档（Save states）

- 随时保存游戏进度  
- 立即恢复  

---

## 🎮 作弊（Cheats）

- 内置作弊代码  
- 可下载作弊文件  

---

## 🎨 滤镜（Shaders）

- CRT 屏幕效果（老电视）  
- 图像平滑处理  

---

## 📁 自动扫描

- 按主机分类整理游戏  
- 自动创建播放列表  

---

# 📂 重要文件结构（PS3）

👉 非常重要，必须正确设置：  
`/dev_hdd0/game/RETROARCH/USRDIR/`

你需要：

- `cores` → 模拟器核心  
- `system` → BIOS 文件  
- ROM → 可放在任意位置（例如：`/roms/nes`）

---

# ⚠️ 常见问题

## ❌ “0/3 scanning”

👉 可能原因：

- 文件夹路径错误  
- core 不正确  
- 文件名称不正确  

---

## ❌ 黑屏

👉 可能原因：

- core 不正确  
- 缺少 BIOS  
- 游戏不兼容  

---

## ❌ 没有作弊选项

👉 解决方法：

- 下载作弊数据库  
- 或手动添加文件
