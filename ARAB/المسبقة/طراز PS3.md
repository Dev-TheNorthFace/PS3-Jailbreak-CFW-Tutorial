# ✅ موديلات PS3 المتوافقة مع CFW (Custom Firmware)

## 🟢 1. PS3 FAT (النسخة الكبيرة)

👉 **جميع أجهزة FAT متوافقة مع CFW**

### الموديلات:
- CECHAxx  
- CECHBxx  
- CECHCxx  
- CECHExx  
- CECHGxx إلى CECHQxx  

✔️ متوافقة بنسبة 100%  
✔️ لا يهم إصدار النظام  

---

## 🟢 2. PS3 Slim المتوافقة مع CFW

👉 **متوافقة بنسبة 100%:**
- CECH-20XX  
- CECH-21XX  

✔️ دائمًا متوافقة مع CFW  

---

## 🟡 3. PS3 Slim (متوافقة جزئيًا)

👉 **CECH-25XX (انتباه ⚠️)**

✔️ متوافقة **فقط إذا:**
- الحد الأدنى للفيرموير ≤ 3.56  
- تم تصنيعها قبل أبريل 2011  
- رمز التاريخ: 9 / 0 / 1A  

❌ **غير متوافقة إذا:**
- MinVerChk = 3.60+  

📌 لذلك → يجب التحقق قبل ذلك
