# 🎮 RetroArch — ¿Qué es?

**RetroArch** es una aplicación en PS3 con jailbreak que permite **jugar a decenas de consolas retro con un solo programa**.

👉 En resumen:  
una **“consola universal” para juegos retro**

---

# 🔥 ¿Para qué sirve?

## 🕹️ 1. Emular muchas consolas

Con RetroArch puedes jugar a:

- NES  
- SNES  
- Game Boy / Game Boy Advance  
- Sega Genesis  
- Game Gear  
- Atari  
- PlayStation 1  
- Arcade (MAME, FBNeo)  

👉 Y muchas más

---

## 🧠 2. Funciona con “cores”

👉 Un **core = un emulador**

### Ejemplos:

- SNES → **Snes9x**  
- NES → **FCEUmm**  
- GBA → **VBA Next**  
- Arcade → **FBNeo**  

👉 Debes **cargar un core antes de iniciar un juego**

---

# ⚙️ 3. Funciones muy potentes

## ⏪ Guardado rápido (Save States)

- Guardar en cualquier momento  
- Continuar instantáneamente  

---

## 🎮 Cheats

- Códigos de trucos integrados  
- Descargar cheats  

---

## 🎨 Filtros (Shaders)

- Efecto CRT (televisión antigua)  
- Suavizado gráfico  

---

## 📁 Escaneo automático

- Organiza los juegos por consola  
- Crea listas automáticamente  

---

# 📂 Estructura importante (PS3)

👉 Muy importante para que funcione:  
`/dev_hdd0/game/RETROARCH/USRDIR/`

Debes tener:

- `cores` → emuladores  
- `system` → BIOS  
- tus ROMs → donde quieras (ej: `/roms/nes`)  

---

# ⚠️ Problemas comunes

## ❌ “0/3 scanning”

👉 Posibles causas:

- Carpeta incorrecta  
- Core incorrecto  
- Nombre de archivo incorrecto  

---

## ❌ Pantalla negra

👉 Posibles causas:

- Core incorrecto  
- BIOS faltante  
- Juego incompatible  

---

## ❌ Sin cheats

👉 Solución:

- Descargar la base de datos de cheats  
- O añadir los archivos manualmente
