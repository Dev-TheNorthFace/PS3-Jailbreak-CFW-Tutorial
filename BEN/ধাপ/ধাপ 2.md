# 🛠️ ধাপ ২ : ব্রাউজার প্রস্তুত করা এবং PS3 Toolset ব্যবহার

---

## 🌐 1. ইন্টারনেট ব্রাউজার খোলা

- যান **Network → Internet Browser**

---

## 🔧 2. ব্রাউজার রিসেট করা (গুরুত্বপূর্ণ)

**🔺 (Triangle)** চাপুন → **Tools**, তারপর সেট করুন :

- **Cookies** → Allow  
- **JavaScript** → Enable  
- **Confirm Browser Close** → Off  
- **Homepage** → Default Page  

এরপর :

- **Delete Cookies** → Yes  
- **Clear Search History** → Yes  
- **Clear Cache** → Yes  
- **Delete Authentication Info** → Yes  

👉 ব্রাউজার সম্পূর্ণ বন্ধ করুন, তারপর **আবার চালু করুন**

---

## 🔎 3. PS3 Toolset ওয়েবসাইটে প্রবেশ

1. **Start** চাপুন  
2. লিখুন : **ps3toolset.com**  
3. সার্চ দিন  

---

## ⚠️ যদি “Update your browser” মেসেজ আসে

যাচাই করুন :

1. ঠিকভাবে URL লিখেছেন (কোনো extra space নেই)  
2. আপনি **Start** ব্যবহার করেছেন, টুলবার না  

👉 না হলে আবার ব্রাউজার রিসেট করুন  

---

## ⏳ 4. ওয়েবসাইট লোড হওয়া

- লোড হতে **১–২ মিনিট** লাগতে পারে  
- উপরে ডানদিকে কিছু মেসেজ আসবে → **এটা স্বাভাবিক**  

👉 লোড হওয়ার সময় কিছু চাপবেন না  

---

## 📜 5. স্ক্রিপ্ট অনুমতি দেওয়া

- যদি স্ক্রিপ্ট ইনস্টল করতে বলে → **Yes**  
- যদি crypto donation মেসেজ আসে → **No**  

---

## 🔁 যদি সাইট কাজ না করে

- ব্রাউজার বন্ধ করুন  
- আবার সম্পূর্ণ রিসেট করুন  
- আবার চেষ্টা করুন  

---

## 🧠 6. CFW compatibility চেক করা

1. **System Manager**-এ ক্লিক করুন  
2. দেখুন :

- ✅ **CFW Compatible PS3** → Compatible  
- ❌ লাল চিহ্ন → Compatible না  

---

## 💾 7. Flash Memory Backup (খুবই গুরুত্বপূর্ণ)

1. USB লাগান (সবচেয়ে ডান পোর্টে, ডিস্ক ড্রাইভের পাশে)  
2. ক্লিক করুন :

- **Flash Memory → Save Flash Memory Backup**  
- **dev_usb000** নির্বাচন করুন  
- **Save** চাপুন  

👉 শেষ হলে **Close** করুন  

📌 USB-তে **dump.bin** নামে ফাইল তৈরি হবে  
👉 এটি নিরাপদে রাখুন (backup হিসেবে)

---

## ⚙️ 8. CFW Patch প্রয়োগ করা

### 1. Patch লোড করা

- **Flash Memory Patch → Load Patch via HTTP**  
- অপেক্ষা করুন → **Close**  

---

### 2. Patch প্রয়োগ

- **Flash Memory Patch → Apply Loaded Patch**  

⚠️ সতর্কতা মেসেজ :

> আপনি আপনার কনসোলের জন্য নিজেই দায়ী।  
> ভুল করলে সমস্যা হতে পারে।

👉 **Yes** চাপুন  

- শেষ হওয়া পর্যন্ত অপেক্ষা করুন  
- **Close** চাপুন  

---
