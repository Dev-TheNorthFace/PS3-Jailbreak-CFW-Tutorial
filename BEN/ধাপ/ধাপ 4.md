# 🛠️ ধাপ ৪ : টুলস ইনস্টল + PS3 কাস্টমাইজ

---

# 🎮 ১. PKG ম্যানেজার বুঝুন

**Game** ট্যাবে আপনি পাবেন :

- 📁 **PlayStation 3** (এখানে প্রয়োজন নেই)  
- 📁 **Package Manager** (গুরুত্বপূর্ণ)  

---

## 📦 PKG ফাইল কী?

একটি **.PKG** ফাইল PS3-এ ব্যবহৃত হয় :

- গেম ইনস্টল করতে  
- অ্যাপ্লিকেশন (homebrew) ইনস্টল করতে  
- টুলস ইনস্টল করতে  

👉 এটি Windows-এর **.exe ফাইলের মতো**  

---

## 📂 "Package Manager" এর ভিতরে

আপনি পাবেন :

### 📁 PlayStation Network Content

- কিছু নির্দিষ্ট কনটেন্টের জন্য ব্যবহৃত  
- 👉 এখানে আমাদের জন্য গুরুত্বপূর্ণ নয়  

---

### 📁 Install Package Files

👉 সবচেয়ে গুরুত্বপূর্ণ

এই ফোল্ডারে :

- **Standard** → USB থেকে PKG ইনস্টল করুন ✅  
- অন্যান্য অপশন → কম গুরুত্বপূর্ণ  

---

### 🗑️ Delete Package Files

- ইনস্টল করা PKG ফাইল মুছে ফেলতে ব্যবহৃত  

---

# 📥 ২. প্রয়োজনীয় সফটওয়্যার ডাউনলোড

**Software (GitHub)** ফোল্ডারে গিয়ে **Liens.md** ফাইলে ক্লিক করুন, তারপর `.pkg` ফাইল ডাউনলোডের জন্য লিঙ্কগুলোতে ক্লিক করুন।

- Apollo Save Tool  
- ManaGunZ  
- MultiMAN  
- PS2 Classics  
- PS2CONFIG  
- RetroArch  
- ROMS PKG  
- webMAN MOD  

---

|----------|--------|

# 🎨 ৩. PS3 থিম ইনস্টল

## 📥 ১. থিম ডাউনলোড

- প্রয়োজনীয় ফরম্যাট : `.p3t`  

👉 উদাহরণ সাইট :  
http://www.ps3-themes.com  

---

## 📂 ২. USB প্রস্তুত করুন

আপনার USB-তে তৈরি করুন :

PS3/  
└── THEME/  
  └── NOMDUTHEME.p3t  

---

## 🔌 ৩. PS3-এ ইনস্টল

1. USB সংযুক্ত করুন (ডানদিকের পোর্ট)  
2. যান : Settings → Theme Settings → Theme  
3. নির্বাচন করুন :

- **Install**  
- **USB (usb000)**  

4. আপনার থিম নির্বাচন করুন  
5. **Apply** ক্লিক করুন
