## 💻 ২. কম্পিউটারে USB প্রস্তুত করুন

1. আপনার USB ড্রাইভটি কম্পিউটারে সংযুক্ত করুন  
2. USB ড্রাইভটি খুলুন  
3. এই ফাইলটি খুঁজুন : **dump.bin**  
4. এটি আপনার কম্পিউটারে কপি করুন (নিরাপদে রাখুন)  

📌 এই ফাইলটি আপনার **ফ্ল্যাশ মেমরির ব্যাকআপ** (সমস্যা হলে খুব গুরুত্বপূর্ণ)

## 📥 ৩. CFW (Evilnat) ডাউনলোড করুন

👉 গাইডের GitHub-এ গিয়ে **Software** ফোল্ডারে যান  

আপনি একটি ফাইল দেখবেন যার নাম **Liens.md**। সেটিতে ক্লিক করুন, তারপর আপনার সিস্টেমের ভার্সনের জন্য উপযুক্ত লিঙ্কে ক্লিক করুন।

### 📌 সঠিক ভার্সন নির্বাচন করুন :

- **4.88** → যদি আপনার PS3 **4.85 থেকে 4.88** এর মধ্যে থাকে  
- **4.92** → যদি আপনার PS3 **4.90 থেকে 4.92** এর মধ্যে থাকে

## 🔌 ৫. USB PS3-এ সংযুক্ত করুন

## ⚙️ ৬. CFW আপডেট ইনস্টল করুন

1. যান : Settings → System Update  
2. নির্বাচন করুন : **Update via Storage Media**  
3. সব ঠিক থাকলে দেখাবে : Version : Evilnat (PEX বা অনুরূপ)  

👉 এটি দেখালে = ✅ ঠিক আছে

## ▶️ ৭. ইনস্টলেশন শুরু করুন
