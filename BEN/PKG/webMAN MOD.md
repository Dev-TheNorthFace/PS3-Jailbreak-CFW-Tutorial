# 🌐 webMAN MOD — এটা কী?

**webMAN MOD** হলো PS3 jailbreak-এর জন্য একটি **খুবই শক্তিশালী plugin**,  
যার মাধ্যমে আপনি **PS3 menu (XMB) থেকেই সরাসরি গেম চালাতে পারবেন**, কোনো app খোলা ছাড়াই।

👉 সহজভাবে :  
সবকিছু **কনসোলের ভিতরেই integrated**

---

# 🔥 এটি কী কাজে লাগে?

## 🎮 1. গেম দ্রুত চালানো

Compatible :

- **PS3 গেম** (ISO / folder)  
- **PS2 / PS1 / PSP গেম**  

👉 এগুলো দেখা যাবে :
Game → webMAN Games

✔️ ক্লিক → mount → play  

---

## ⚡ 2. খুব দ্রুত

👉 MultiMAN-এর তুলনায় :

- কোনো লম্বা loading নেই  
- কোনো জটিল menu navigation নেই  

---

## 🌡️ 3. Fan control (খুব গুরুত্বপূর্ণ)

webMAN দিয়ে :

- fan speed নিয়ন্ত্রণ করা যায়  
- overheating এড়ানো যায় 🔥  

আপনি করতে পারবেন :

- Auto mode  
- Manual control  

👉 পরামর্শ :

- CPU temperature রাখুন **60–68°C**

---

## 🌐 4. Web interface (PC / ফোন)

Browser দিয়ে PS3 access করতে পারবেন :
http://192.168.X.X

আপনি করতে পারবেন :

- গেম চালানো  
- temperature দেখা  
- console বন্ধ করা  
- file manage করা  

---

## 📂 5. ব্যবহৃত folder

webMAN নিজে থেকেই detect করে :
/dev_hdd0/PS3ISO/
/dev_hdd0/PS2ISO/
/dev_hdd0/PSXISO/
/dev_hdd0/PSPISO/

👉 এখানে গেম রাখলেই → auto detect হবে  

---

## 🧠 6. Auto scan

- গেম নিজে detect করে  
- list update করে  
- কোনো config দরকার নেই
