# 🎮 RetroArch — এটা কী?

**RetroArch** হলো PS3 jailbreak-এর একটি অ্যাপ, যার মাধ্যমে  
আপনি **একটি প্রোগ্রাম দিয়েই অনেকগুলো retro console-এর গেম খেলতে পারবেন**।

👉 সহজভাবে :  
একটি **“universal console” retro গেমের জন্য**

---

# 🔥 এটি কী কাজে লাগে?

## 🕹️ 1. অনেক console emulate করা

RetroArch দিয়ে আপনি খেলতে পারবেন :

- NES  
- SNES  
- Game Boy / Game Boy Advance  
- Sega Genesis  
- Game Gear  
- Atari  
- PlayStation 1  
- Arcade (MAME, FBNeo)  

👉 আরও অনেক কিছু  

---

## 🧠 2. “Core” সিস্টেম

👉 একটি **core = একটি emulator**

### উদাহরণ :

- SNES → **Snes9x**  
- NES → **FCEUmm**  
- GBA → **VBA Next**  
- Arcade → **FBNeo**  

👉 গেম চালানোর আগে অবশ্যই **core load করতে হবে**

---

# ⚙️ 3. শক্তিশালী ফিচার

## ⏪ Save States

- যেকোনো সময় save করা  
- সাথে সাথে resume করা

## 🎮 Cheats

## 🎨 Shaders (Filters)

- CRT effect (পুরোনো TV look)  
- graphics smooth করা

## 📁 Auto Scan

- গেমগুলো console অনুযায়ী সাজায়  
- automatic playlist তৈরি করে  

---

# 📂 গুরুত্বপূর্ণ ফোল্ডার (PS3)

👉 কাজ করার জন্য গুরুত্বপূর্ণ path :
/dev_hdd0/game/RETROARCH/USRDIR/


আপনার থাকতে হবে :

- `cores` → emulator  
- `system` → BIOS  
- ROMs → যেকোনো জায়গায় (যেমন `/roms/nes`)
