# 🔥 এটি কী কাজে লাগে?

MultiMAN দিয়ে চালাতে পারবেন :

- **PS3 গেম** (ISO বা folder format)  
- **PS1 গেম**  
- **PS2 গেম** (কিছু সীমাবদ্ধতা সহ)  

👉 চালানো যায় :

- Internal hard drive থেকে  
- USB ড্রাইভ থেকে

MultiMAN একটি file explorer-এর মতো কাজ করে :

আপনি করতে পারবেন :

- গেম কপি / পেস্ট  
- ফাইল মুভ করা  
- ফাইল ডিলিট / রিনেম করা  

অ্যাক্সেস করতে পারবেন :
