- CECHAxx  
- CECHBxx  
- CECHCxx  
- CECHExx  
- CECHGxx থেকে CECHQxx

## 🟡 3. PS3 Slim (আংশিক সমর্থিত)

👉 **CECH-25XX (সতর্কতা ⚠️)**

✔️ শুধুমাত্র তখনই কাজ করবে যদি :

- Minimum firmware ≤ 3.56  
- এপ্রিল ২০১১ এর আগে তৈরি  
- Date code : 9 / 0 / 1A  

❌ কাজ করবে না যদি :

- MinVerChk = 3.60+  

📌 তাই আগে অবশ্যই যাচাই করতে হবে
