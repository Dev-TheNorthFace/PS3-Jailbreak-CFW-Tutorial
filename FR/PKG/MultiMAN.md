# 🎮 MultiMAN — c’est quoi ?

**MultiMAN** est le **manager de jeux le plus connu** sur PS3 jailbreak.  
C’est généralement le **premier outil installé** sur une PS3 en **CFW ou HEN**.

---

# 🔥 À quoi ça sert ?

## 📀 1. Lancer des jeux (fonction principale)

MultiMAN permet de lancer :

- Jeux **PS3** (ISO ou format dossier)  
- Jeux **PS1**  
- Jeux **PS2** (avec certaines limitations)  

👉 Depuis :

- Disque dur interne  
- Clé USB  

---

## 📂 2. Gestionnaire de fichiers (TRÈS IMPORTANT)

MultiMAN agit comme un explorateur de fichiers :

Tu peux :

- Copier / coller des jeux  
- Déplacer des fichiers  
- Supprimer / renommer  

Accès aux dossiers :

- `/dev_hdd0`  
- USB  
- Réseau  

👉 C’est comme un **explorateur Windows sur PS3**

---

## 🖼️ 3. Interface

- Style **XMB (menu PS3)**  
- Mode **Coverflow (jaquettes)**  
- Interface simple mais un peu ancienne  

---

## 🌐 4. Transfert de jeux (FTP)

MultiMAN permet :

- Connexion via **FileZilla**  
- Transfert de jeux **PC → PS3**  

👉 Très utile pour installer des jeux rapidement

---

## 📦 5. Installation de PKG

Tu peux :

- Installer des **homebrews**  
- Installer des **jeux en PKG**  

---

## 💿 6. Copier des disques (Rip)

Tu peux :

- Insérer un disque de jeu  
- Le copier directement sur le disque dur  

👉 Plus besoin du disque après

---

- ✔️ Parfait pour débutants  
- ✔️ Indispensable
