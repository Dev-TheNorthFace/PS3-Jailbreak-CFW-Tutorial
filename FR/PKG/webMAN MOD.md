# 🌐 webMAN MOD — c’est quoi ?

**webMAN MOD** est un **plugin ultra puissant** pour PS3 jailbreak qui permet de  
**lancer tes jeux directement depuis le menu PS3 (XMB)** sans ouvrir d’application.

👉 En gros :  
tout est **intégré directement dans la console**

---

# 🔥 À quoi ça sert ?

## 🎮 1. Lancer tes jeux instantanément

Compatible avec :

- Jeux **PS3** (ISO / dossier)  
- Jeux **PS2 / PS1 / PSP**  

👉 Ils apparaissent dans : Jeu → webMAN Games


✔️ Tu cliques → ça monte → tu lances  

---

## ⚡ 2. Ultra rapide

👉 Contrairement à MultiMAN :

- Pas de chargement long  
- Pas de navigation compliquée  

---

## 🌡️ 3. Contrôle du ventilateur (TRÈS IMPORTANT)

webMAN permet de :

- Contrôler la vitesse du ventilateur  
- Éviter la surchauffe 🔥  

Tu peux :

- Mode automatique  
- Réglage manuel  

👉 Conseil :

- Garder le CPU autour de **60–68°C**

---

## 🌐 4. Interface web (PC / téléphone)

Tu peux accéder à ta PS3 via navigateur : http://192.168.X.X


Tu peux :

- Lancer des jeux  
- Voir la température  
- Éteindre la console  
- Gérer les fichiers  

---

## 📂 5. Dossiers utilisés

webMAN détecte automatiquement :

- /dev_hdd0/PS3ISO/
- /dev_hdd0/PS2ISO/
- /dev_hdd0/PSXISO/
- /dev_hdd0/PSPISO/


👉 Tu mets tes jeux ici → ils apparaissent automatiquement  

---

## 🧠 6. Scan automatique

- Détecte les jeux automatiquement  
- Met à jour la liste  
- Aucun réglage nécessaire
