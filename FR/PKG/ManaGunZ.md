# 🔥 ManaGunZ — c’est quoi ?

**ManaGunZ** est un **manager de jeux avancé** sur PS3 jailbreak (**CFW surtout**).  
Il est particulièrement connu pour ses **patchs PS2 automatiques** et ses outils avancés.

---

# 🧠 À quoi ça sert concrètement ?

## 🎮 1. Gestion des jeux (PS1 / PS2 / PS3)

ManaGunZ permet de :

- Lancer des jeux **PS3 (ISO / dossier)**  
- Lancer des jeux **PS2 (ISO)**  
- Lancer des jeux **PS1**  

👉 Alternative à **multiMAN**, mais plus technique

---

## 🧠 2. Patchs automatiques PS2 (GROS AVANTAGE)

👉 C’est là où ManaGunZ est 🔥

Il permet de :

- Appliquer automatiquement des **configs de compatibilité PS2**
- Corriger :
  - Bugs graphiques  
  - Écrans noirs  
  - Problèmes de performance  

---

## ⚙️ 3. Modifier les configs PS2

Tu peux aussi :

- Forcer la résolution  
- Activer / désactiver certains patchs  
- Tester différentes configurations  

👉 Utile pour les jeux PS2 difficiles à faire fonctionner

---

## 📦 4. Gestion avancée des fichiers

- Navigateur de fichiers intégré  
- Copier / coller des jeux  
- Gestion des dossiers ISO  

---

## 🎨 5. Interface personnalisable

- Jaquettes automatiques  
- Interface propre et moderne  
- Meilleure organisation des jeux  

---

## 🧪 6. Outils avancés (pour utilisateurs avancés)

- Analyse des jeux  
- Gestion des modules  
- Options de debug  

---

# 💡 Exemple concret

Un jeu PS2 ne marche pas :

1. Tu lances le jeu avec ManaGunZ  
2. Il applique automatiquement une config  
3. Si ça ne marche pas :
   - Tu modifies les patchs manuellement  

👉 Et souvent → le jeu fonctionne
