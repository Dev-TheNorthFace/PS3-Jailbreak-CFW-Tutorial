# 📌 Informations générales

Beaucoup de personnes disent que pour jailbreak la console, il est important d’avoir la bonne version du firmware.  
Pour être honnête avec vous, non : du moment que le modèle de votre console est compatible CFW, peu importe la version du logiciel, cela va fonctionner pareil.

---

## 🔑 Clé USB

Pour la clé USB, quand vous voulez la connecter sur votre PlayStation 3, mettez-la dans le port USB **le plus à droite** (le plus proche du lecteur).  
Sinon, la clé USB peut ne pas être lue par la console.

---

## 🔄 Réinitialisation

Si vous achetez une PlayStation 3 ou si vous en avez une qui traîne depuis longtemps, je vous conseille fortement de la **réinitialiser**.

👉 Pourquoi ?  
Pour éviter les bugs ou problèmes.

---

## ⬆️ Mise à jour du firmware

Si votre console est sur un firmware trop ancien, je vous conseille de la **mettre à jour** :

- ✔️ Internet fonctionne mieux  
- ✔️ Moins de problèmes avec le CFW  
- ✔️ Dernière version : **4.92**

---

## 🌐 Connexion Internet

Pour télécharger les jeux plus vite ou accéder à Internet plus rapidement depuis votre PS3, je vous conseille d’utiliser un **câble Ethernet** branché à l’arrière de la console.

👉 Avantages :
- Meilleure vitesse de connexion  
- Moins de problèmes avec le navigateur  

---

## 💾 Stockage (disque dur)

Les vieilles PS3 n’ont souvent pas assez d’espace.

👉 Je vous conseille d’acheter un disque dur :
- 256 Go  
- 512 Go  
- 1 To  

📌 **Important :**  
Si vous faites le jailbreak et que vous changez le disque dur, il faudra **refaire le jailbreak** après l’installation.

👉 Mais :
- Ce sera plus rapide que la première fois  
- Le Patch Memory n’est pas sur le disque dur, mais directement sur la carte mère de la console  

---

## ⚠️ PSN (Très important)

❗ Ne vous connectez **jamais** au PSN avec une PS3 jailbreak.

👉 Risques :
- ❌ Bannissement de la console  
- ❌ Bannissement du compte PSN
