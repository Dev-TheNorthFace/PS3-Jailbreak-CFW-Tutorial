# 💾 Changer le disque dur de votre PS3 (CFW)

Si vous remplacez votre disque dur par un modèle avec plus de stockage, voici ce qu’il faut faire 👇

---

## 🔌 1. Préparer la clé USB

- Prenez une clé USB formatée en **FAT32**
- Ajoutez le fichier de mise à jour CFW correspondant à votre version (ex : **4.92 Evilnat**)

Structure à respecter :
PS3/
└── UPDATE/
└── PS3UPDAT.PUP


---

## ⚙️ 2. Installer le nouveau disque dur

- Remplacez le disque dur de la PS3  
- Allumez la console  

👉 La PS3 demandera automatiquement une **réinstallation du système**

---

## 🔄 3. Réinstaller le CFW

1. Branchez la clé USB  
2. Suivez les instructions à l’écran  
3. Installez le firmware (CFW)

---

## 📦 4. Réinstaller les logiciels

⚠️ Important :

- Tous les homebrews (MultiMAN, webMAN, etc.) seront supprimés  
- Vous devrez réinstaller vos **PKG**

---

## 🎮 5. Réinstaller les jeux

- Les jeux présents sur l’ancien disque dur sont perdus  
- Vous devrez :

  - Re-transférer vos jeux  
  - Réinstaller vos PKG  
  - Remettre vos ROMs  

---

## 💡 Astuce (recommandé)

👉 Avant de changer de disque dur :

- Sauvegardez vos données importantes :
  - Saves (Apollo)  
  - Jeux  
  - Fichiers  

---

# ⚠️ Important

- ❗ Toujours utiliser la **même version ou plus récente** du CFW  
- ❗ Ne jamais installer un firmware incompatible  
- ❗ Vérifier que votre PS3 est toujours compatible CFW  

---

# 🎯 Résumé simple

👉 Changement de disque dur =

- Réinstaller le CFW  
- Réinstaller les PKG  
- Réinstaller les jeux
