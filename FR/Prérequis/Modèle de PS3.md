# ✅ Modèles PS3 compatibles CFW (Custom Firmware)

## 🟢 1. PS3 FAT (grosse PS3)

👉 **TOUTES les FAT sont compatibles CFW**

### Modèles :
- CECHAxx
- CECHBxx
- CECHCxx
- CECHExx
- CECHGxx à CECHQxx

✔️ 100% compatibles  
✔️ Peu importe la version  

---

## 🟢 2. PS3 Slim compatibles CFW

👉 **100% compatibles :**
- CECH-20XX
- CECH-21XX

✔️ Toujours OK pour CFW  

---

## 🟡 3. PS3 Slim (partiellement compatibles)

👉 **CECH-25XX (ATTENTION ⚠️)**

✔️ Compatible **SEULEMENT SI :**
- Minimum firmware ≤ 3.56  
- Fabriquée avant avril 2011  
- Date code : 9 / 0 / 1A  

❌ **Pas compatible si :**
- MinVerChk = 3.60+  

📌 Donc → faut vérifier avant  

---

# ❌ Modèles PS3 NON compatibles CFW

## 🔴 1. PS3 Slim récentes

👉 **CECH-30XX**

❌ CFW IMPOSSIBLE  
✔️ Seulement HEN  

---

## 🔴 2. PS3 Super Slim

👉 **TOUTES les Super Slim :**
