# 🛠️ Étape 2 : Préparer le navigateur et utiliser PS3 Toolset

---

## 🌐 1. Ouvrir le navigateur internet

- Allez dans **Réseau → Navigateur Internet**

---

## 🔧 2. Réinitialiser le navigateur (IMPORTANT)

Appuyez sur **🔺 (Triangle)** → **Outils**, puis configurez :

- **Cookies** → Permettre  
- **JavaScript** → Activé  
- **Confirmer fermeture navigateur** → Non  
- **Page d’accueil** → Utiliser page par défaut  

Ensuite :

- **Effacer les cookies** → Oui  
- **Supprimer l’historique de recherche** → Oui  
- **Effacer le cache** → Oui  
- **Effacer les informations d’authentification** → Oui  

👉 Fermez complètement le navigateur, puis **ouvrez-le à nouveau**

---

## 🔎 3. Accéder au site PS3 Toolset

1. Appuyez sur **Start**
2. Entrez l’adresse : **ps3toolset.com**
3. Lancez la recherche

---

## ⚠️ Si le message "Mettre à jour votre navigateur" apparaît

Vérifiez :

1. L’adresse est bien écrite (aucun espace en trop)  
2. Vous avez utilisé la barre d’outils pour votre rechercher et non **Start**

👉 Sinon, recommencez la réinitialisation du navigateur

---

## ⏳ 4. Chargement du site

- Le site peut prendre **1 à 2 minutes** à charger  
- Des messages apparaissent en haut à droite → **c’est normal**

👉 Ne touchez à rien pendant le chargement

---

## 📜 5. Autoriser les scripts

- Si le navigateur demande d’installer un script → **Oui**  
- Si un message de don en crypto apparaît → **Non**

---

## 🔁 Si le site ne fonctionne pas

- Fermez le navigateur  
- Refaites la **réinitialisation complète de votre navigateur**  
- Réessayez  

---

## 🧠 6. Vérifier la compatibilité CFW

1. Cliquez sur **System Manager**
2. Regardez :

- ✅ **CFW Compatible PS3** → Compatible  
- ❌ **Croix rouge** → Non compatible  

---

## 💾 7. Sauvegarder la mémoire flash (TRÈS IMPORTANT)

1. Branchez une clé USB dans le **port le plus à droite** (près du lecteur CD)
2. Cliquez sur :

- **Flash Memory → Save Flash Memory Backup**
- Sélectionnez **dev_usb000**
- Cliquez sur **Save**

👉 Attendez la fin, puis **Close**

📌 Un fichier **dump.bin** sera créé sur votre clé USB  
👉 Gardez-le en sécurité (backup en cas de problème)

---

## ⚙️ 8. Appliquer le patch CFW

### 1. Charger le patch

- **Flash Memory Patch → Load Patch via HTTP**
- Attendre → **Close**

### 2. Appliquer le patch

- **Flash Memory Patch → Apply Loaded Patch**

⚠️ Message de confirmation :

> Vous êtes responsable de votre console.  
> Une mauvaise manipulation peut causer un problème.

👉 Cliquez sur **Yes**

- Attendez la fin du processus  
- Cliquez sur **Close**

---

## ✅ 9. Fin de l’étape

- Quittez le navigateur  

---

# 🎉 Étape 2 terminée !

👉 Vous pouvez maintenant passer à **l’étape 3**
