# 🛠️ Étape 3 : Installation du CFW (Evilnat)

---

## 🔌 1. Redémarrer la console

- Éteignez complètement votre PS3  
- Rallumez-la  

---

## 💻 2. Préparer la clé USB sur PC

1. Branchez votre clé USB sur votre ordinateur  
2. Ouvrez la clé USB  
3. Trouvez le fichier : **dump.bin**
4. Copiez-le sur votre PC (garde-le en sécurité)
📌 Ce fichier est une **sauvegarde de votre mémoire flash** (très important en cas de problème)

---

## 📥 3. Télécharger le CFW (Evilnat)

👉 Sur le GitHub du guide, allez dans le dossier **Software**

Vous verrez un fichier appelé **Liens.md**. Cliquez dessus, puis cliquez sur le lien correspondant à la version de votre système.

### 📌 Choisir la bonne version :

- **4.88** → si votre PS3 est entre **4.85 et 4.88**
- **4.92** → si votre PS3 est entre **4.90 et 4.92**

---

## 📂 4. Mettre les fichiers sur la clé USB

Dans le dossier téléchargé, vous aurez :
PS3/
└── UPDATE/
└── PS3UPDAT.PUP


👉 Copiez **le dossier PS3 complet** sur votre clé USB  

✔️ Résultat attendu sur la clé USB :
└── PS3/
└── UPDATE/
└── PS3UPDAT.PUP


---

## 🔌 5. Brancher la clé USB sur la PS3

- Branchez la clé USB dans le **port le plus à droite**
  (le plus proche du lecteur CD)

---

## ⚙️ 6. Installer la mise à jour CFW

1. Allez dans : Paramètres → Mise à jour système
2. Choisissez : **Mettre à jour via support de stockage**
3. Si tout est bon, vous verrez : Version : Evilnat (PEX ou similaire)
👉 Si vous voyez ça = ✅ parfait

---

## ▶️ 7. Lancer l’installation

- Cliquez sur **OK**
- Acceptez toutes les conditions
- Attendez la fin de l’installation
⚠️ Ne touchez à rien pendant la mise à jour

---

## 🎮 8. Vérifier que le jailbreak a fonctionné

Au redémarrage :

- Le logo **PlayStation 3** peut être remplacé par **Evilnat**

👉 Si oui → 🎉 **console jailbreak avec succès !**
